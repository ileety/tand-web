모바일 사용 행태 데이터 제공자(매체)와 사용자(마케터)가
상호 안전하고 효율적으로 협력할 수 있는
모바일 마케팅 데이터 거래 플랫폼을 제공합니다.

About

  • Technology

    모바일 어플리케이션의 사용행태를
    분석하여 사용자의 다음 행동을 예측할 수
    있는 분석 알고리즘을 제공합니다.

  • Big Data

    일 이벤트(event) 10억 회 이상의
    데이터를 처리하여 효율인 검색과 분석이
    가능한 빅데이터 인프라를 제공합니다.

  • Easy to Use

    마케터가 직접 데이터를 검색, 비교,
    분석하여 데이터 기반의 모바일 마케팅
    전략을 수립할 수 있습니다.

Data Partner

귀사의 데이터 자산의 가치를 극대화 하십시오.
매체 사용(Behavior)행태는 소중한 마케팅(타깃팅) 데이터로 활용됩니다.
우수한 데이터 고객은 데이터의 규모(Scale)와 다양성(Diversity)을 요구합니다.

귀사의 데이터 자산과 텐디의 자산을 통합하여 데이터 고객이 만족할 수 있는 규모와 다양성을 제공할 수 있습니다.
